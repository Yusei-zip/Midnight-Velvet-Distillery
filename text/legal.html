Términos de uso

Nocturne Bar es una aplicación de demostración que utiliza la API pública de TheCocktailDB con fines exclusivamente educativos y experimentales.

El usuario reconoce que la información mostrada puede variar sin previo aviso debido a la naturaleza de la fuente de datos externa.

Este proyecto no representa un servicio comercial oficial ni garantiza la exactitud total de las recetas o datos mostrados.

Privacidad

Nocturne Bar no recopila, almacena ni procesa datos personales del usuario.

Únicamente se utiliza el almacenamiento local del navegador (localStorage) para guardar la lista de cócteles favoritos del usuario.

No se comparten datos con terceros ni se utilizan sistemas de seguimiento.

Licencias

Este proyecto utiliza la API pública de TheCocktailDB bajo sus propios términos de uso.

Todo el código del proyecto ha sido desarrollado con fines educativos y de aprendizaje, y puede ser reutilizado con fines no comerciales.
