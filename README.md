# Midnight Velvet Distillery

Explorador de cócteles con estética nocturna y diseño moderno. Este proyecto está construido con HTML, CSS y JavaScript puros, y funciona como una página estática que muestra cócteles, filtros y detalles en modal.

## Estructura

- `index.html` — Página principal
- `css/` — Estilos divididos en módulos
- `js/` — Lógica de la aplicación y renderizado
- `favorito.html` — Página de favoritos
- `text/legal.html` — Texto legal

## Uso

1. Abre `index.html` en tu navegador.
2. Busca cócteles, usa los filtros por ingrediente o usa el botón de cóctel aleatorio.
3. Haz clic en un cóctel para ver los detalles y agregarlo a favoritos.

## Recomendaciones

- Sube el proyecto a GitHub como un repositorio estático.
- Si lo quieres servir localmente, usa cualquier servidor estático simple (por ejemplo, `Live Server` en VS Code).

## Notas

- Ya está preparado para ser responsivo en dispositivos móviles sin cambiar el layout de escritorio.
- El diseño usa `css/style.css` como archivo principal de importación.
